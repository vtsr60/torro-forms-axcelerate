Torro Forms Plugin to create contact in aXcelerate SMS.

== Description ==

> This is an extension for the [Torro Forms](https://wordpress.org/plugins/torro-forms/) plugin which allows to create contact in aXcelerate SMS on form submission. You need to have it enabled in order to run this plugin. API and Web Service token for aXcelerate SMS. You can create the tokens in the aXcelerate admin section under Settings &#62; System Settings &#62; [Web & Other Integrations](https://admin.axcelerate.com.au/management/dashboard/index.cfm?tab=WEB).

== Installation ==

1. Upload the entire `torro-forms-axcelerate` folder to the `/wp-content/plugins/` directory or download it through the WordPress backend.
2. Activate the plugin through the 'Plugins' menu in WordPress.
